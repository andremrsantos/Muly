== Muly
Este aplicativo foi desenvolvido para auxiliar no aprendizado do framework Ruby on Rails para o 1o Rails Girls de Bélem, PA.

Um aplicativo simples, onde se cadastra as informações referentes a uma banda e seus álbuns. Ele explora artificios da linguagem como: 
- Geradores
- Relacionamentos
- Helpers
- Rotas
- Controles
- Outros...
